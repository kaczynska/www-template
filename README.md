# www-template

## Description
Website template.

## Technologies
* HTML
* CSS/SASS(scss)
* Java Script 
* Node.js
* Gulp

## How to use
* install all dependencies (make sure nodejs with npm is installed on your machine): npm install
* run: gulp (localhost:3000)
* build: gulp build
* clean: gulp clean
